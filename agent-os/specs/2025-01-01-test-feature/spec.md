# Test Feature Specification

## Overview

Add a simple calculator module to the project with basic arithmetic operations.

## Requirements

1. Create a `calculator.js` file in the `src/` directory
2. Implement the following functions:
   - `add(a, b)` - returns sum of two numbers
   - `subtract(a, b)` - returns difference of two numbers
   - `multiply(a, b)` - returns product of two numbers

## Technical Notes

- Use plain JavaScript (no dependencies)
- Export functions for use in other modules
